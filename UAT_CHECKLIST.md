# UAT Checklist — Очередь админ-бота

- [ ] Фильтры очереди показывают только доступные города для городских ролей, пагинация и кнопки навигации работают.
- [ ] Ручное назначение: предупреждения вне смены/при лимите, подтверждение оффера и возврат на карточку после отправки.
- [ ] Автоматический подбор из карточки запускает сервис распределения, пишет лог и корректно обрабатывает ошибки (нет района/кандидатов).
- [ ] Действие «↩️ Вернуть в поиск» снимает назначение, отменяет офферы и добавляет запись в историю статусов.
- [ ] Действие «❌ Отменить» запрашивает причину, хранит её в истории и скрывает кнопки для отменённых/закрытых заявок.

- [ ] Закрытие заказа мастером (NORMAL): после загрузки акта комиссия создаётся один раз, ставка 40% при avg_week_check >= 7000 иначе 50%, сумма совпадает с round(total_sum * rate, 2), дедлайн оплаты = +3 часа.
- [ ] Гарантийный заказ (GUARANTEE) не формирует комиссию: в карточке заказа нет записи, в логах фиксируется `commission_skip: guarantee`.
- [ ] Owner requisites: step-by-step editor for global admin (validations, /skip, /cancel) updates summary and broadcast delivers snapshot to WAIT_PAY masters.

- Финансы: просрочка комиссии переводится watchdog-ом в OVERDUE, мастер блокируется, алерт в ALERTS содержит кнопку adm:f:cm:<cid>.

- Проверить, что мастер в карточке комиссии получает снапшот реквизитов и отдельное сообщение с QR-кодом, а админ в карточке может открыть чеки, ввести сумму оплаты (/skip для значения по умолчанию), подтвердить/отклонить/заблокировать мастера и вернуться к списку без «мертвых» кнопок.
- [ ] Закрытая заявка в карточке показывает кнопку «Гарантийный выезд»; создание создаёт новый заказ type=GUARANTEE с preferred_master, исходная карточка обновляется без кнопки.
- [ ] При отказе/таймауте прежнего мастера по гарантийной заявке мастер блокируется с причиной  `guarantee_refusal`, запись попадает в логи распределения и алерт. 
- [ ] Закрытие гарантийного заказа мастером: сумма не запрашивается, акт обязателен, статус сразу CLOSED и активные комиссии не создаются.
