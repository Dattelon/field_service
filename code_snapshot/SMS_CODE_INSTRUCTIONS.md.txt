# ИНСТРУКЦИЯ: Получение SMS кода

## Вариант 1: Подождать появления кнопки SMS

1. Оставайтесь на странице где просят Code
2. Подождите 30-60 секунд
3. Появится текст "Didn't get the code?" или "Send via SMS"
4. Нажмите на эту ссылку
5. Код придёт по SMS на +79031751130

## Вариант 2: Закрыть и начать заново (часто помогает)

1. Закройте вкладку Telegram Web
2. Откройте новую вкладку
3. Зайдите на: https://web.telegram.org/k/
4. Введите номер: +79031751130
5. Сразу после ввода номера должна быть опция "Use SMS code"

## Вариант 3: Через старую версию Telegram Web

1. Откройте: https://web.telegram.org (БЕЗ /k/)
2. Там обычно сразу даёт выбор SMS или App
3. Выберите SMS
4. Введите код

## Вариант 4: Desktop приложение (100% работает)

Если Web не даёт SMS:
1. Скачать Desktop: https://desktop.telegram.org/
2. Установить и запустить
3. Ввести номер: +79031751130
4. Там ВСЕГДА есть опция получить SMS

## ⏰ Важно:

- Telegram сначала пытается отправить в приложение
- Через 1-2 минуты появляется опция SMS
- НЕ запрашивайте код повторно слишком часто
- Между попытками делайте паузу 2-3 минуты
