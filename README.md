# Field Service – Operations Notes\n\n## Monitoring\n- Heartbeat: both bots send heartbeat: <name> alive every 60 seconds to the logs channel.\n- Alerts: unexpected errors, distribution escalations and overdue commissions go to the alerts channel.\n- Single instance: on 409 conflict the bot writes to logs and exits.\n\n## Backups\n- Use ops/backup_db.sh (Linux/macOS) or ops/backup_db.ps1 (Windows) to create pg_dump -Fc snapshots.\n- Set DATABASE_URL/BACKUP_DIR env vars. Default directory is ackups/.\n- Old backups older than 7 days are removed automatically from the scripts.\n- Recommended schedule: daily at 03:00 server time (cron / Task Scheduler).\n\n
