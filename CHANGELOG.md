# Changelog

## Unreleased
- Просрочка комиссий: watchdog каждые 10 минут переводит WAIT_PAY → OVERDUE, блокирует мастера, логирует событие и шлёт алерт с кнопкой перехода в карточку комиссии.
- Owner requisites are now stored in staff_users; admin bot has a guided editor and broadcast of current details.
- Переработан расчёт комиссий: CommissionService создаёт запись при переходе заказа в PAYMENT (ставка 40% при avg_week_check >= 7000 иначе 50%), дедлайн оплаты +3 часа, копируется снапшот реквизитов владельца и гарантийные заказы пропускаются; обновлены обработчики мастера и тесты.
- Добавлено отображение очереди заявок с фильтрами, пагинацией и управлением навигацией в админ-боте.
- Обновлён ручной подбор мастеров: список показывает статус смены/лимита, добавлено подтверждение и отправка оффера из очереди.
- Автоназначение из карточки заказа вызывает сервис распределения, логирует тик и обрабатывает ошибки (нет района/кандидатов).

- Финансовый UI обновлён: мастер видит снапшот реквизитов и QR, админ может просматривать чеки, вводить фактическую сумму оплаты и управлять подтверждением/блокировкой прямо из карточки комиссии; добавлены тесты и обработка возврата в список.
- Добавлена поддержка гарантийных заявок: из карточки закрытого заказа можно создать гарантийный выезд, мастер закрывает без суммы (только акт), комиссия не формируется, распределение приоритизирует прежнего исполнителя и автоблокирует его при отказе/таймауте.
